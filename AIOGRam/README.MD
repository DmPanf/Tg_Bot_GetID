# Telegram Bot based on AIOGram

- **`python3 -m venv venv`**

# Конфигурация poetry + Docker

- **`poetry init`**
- **`poetry add aiogram python-dotenv configparser`**
- **`docker-compose build`**
- *`docker-compose up`**
